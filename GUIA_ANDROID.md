## 🆘 PROBLEMAS COMUNS

### ❌ "GitHub/Vercel não autoriza no Android"

**Solução:**
```
1. Limpe cache do navegador
2. Tente em aba anônima
3. Use Chrome (funciona melhor)
4. Ou use Render/GitHub Pages (mais simples)
```

### ❌ "Deploy dá erro"

**Solução:**
```
1. Certifique que uploadou TODOS os 11 arquivos
2. Verifique se index.html está na raiz
3. Aguarde 5 minutos e tente de novo
```

### ❌ "Ranking não funciona"

**Solução:**
```
1. Verificar se executou SQL no Supabase
2. Abrir Console (F12) e ver erros
3. Certificar que credenciais estão corretas
```
